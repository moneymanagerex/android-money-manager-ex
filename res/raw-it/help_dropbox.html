MoneyManagerEx for Android: sincronizzazione il Database con MoneyManagerEx per Windows/Linux/Mac utilizzando il servizio Dropbox.

Prerequisiti: disporre di un account dropbox. Se ancora non lo si possiede, andare su www.dropbox.com e crearne uno nuovo.

  • Collegare MoneyManagerEx per Android con Dropbox. Selezionare "Collegao a Dropbox". Se il dispositivo ha installato l'app di Dropbox, si apre automaticamente e ti chiede di autorizzare MoneyManagerEx per Android accedere il tuo Dropbox. Se non si ha installato l'app di Dropbox sul tuo dispositivo, Android apre il browser sulla pagina di login di Dropbox. Dopo il login, verrà chiesto di autorizzare MoneyManagerEx forAndroid accedere al tuo account Dropbox.
  • Se si autorizza MoneyManagerEx for Android, la prima volta verrà creata automaticamente dal servizio Dropbox, la cartella / applicazioni/MoneyManagerEx Mobile all'interno del vostro spazio Dropbox. La cartella non va mai creata manualmente!

Altre opzioni

Domande frequenti

    Uso già un database per la versione desktop
  • Se hai gi à un database (mmb) della versione Desktop MoneyManagerEx, devi copiare o spostare il file (s) all'interno della cartella /apps/MoneyManagerEx Mobile (consigliamo di utilizzare il computer per effettuare questa operazione).
  • Dalle impostazioni di Dropbox del MoneyManagerEx for Android, cliccare su "File Dropbox". Applicazione aprirà una nuova finestra con tutti i file nella cartella /apps/MoneyManagerEx Mobile. Selezionare il file che si desidera connettere e tornare indietro.
  • Una volta tornato nelle impostazioni Dropbox di MoneyManagerEx for Android, dopo circa 10 secondi si avvierà il download del file selezionato, è possibile vedere la notifica che nella barra di stato nel vostro dispositivo. Quando il download è terminato, il dispositivo ti avvisa che il file è pronto all'uso e si deve cliccare su "Apri Database" per completare il collegamento tra MoneyManagerEx for Android e Dropbox.

    • Non ho mai usato la versione desktop e voglio sincronizzare il database nel dispositivo
  • Se non avete già un file di mmb MoneyManagerEx versione Desktop, dovete cliccare su "Upload a Dropbox database." Dopo circa 10 secondi si avvia Caricamento database Dropbox attualmente in uso sul vostro spazio Dropbox. Alla fine del dispositivo notificherà il completamento dell'upload nella cartella /Apps/MoneyManagerEx Mobile , troverete un file denominato data.mmb.

    • Altre opzioni
  • Dopo aver effettuato il collegamento account e file, MoneyManagerEx for Android automaticamente sincronizza con Dropbox ogni x minuti impostato nell'impostazione "Intervallo di sincronizzazione". Se non si desidera sincronizzare automaticamente impostato "Intervallo Sincronizzazione" a Mai. A questo punto la sincronizzazione dovrà essere effettuata manualmente.
  • È importante impostare "Upload database immediatamente dopo cambiamento". Questa configurazione permette a MoneyManagerEx for Android di caricare il database su Dropbox, immediatamente dopo aver eseguito una modifica.
  • Utilizzare "Download database da Dropbox" o "Upload a Dropbox" per forzare l'operazione di Download o Upload del database, in caso di malfunzionamenti.
  • Quando collegato il tuo account Dropbox e il file nel Drawer si attiva una nuova voce di menù che vi permette la sincronizzazione.

    • Domande frequenti
  • D. Non riesco a collegarmi con Dropbox, o MoneyManagerEx a sincronizzare?
    R. La prima cosa che devi fare è inviarci una mail android.money.manager.ex@gmail.com descrivendo il problema. Nel frattempo, se volete utilizzare MoneyManagerEx per Android ci vorrà utilizzare altre applicazioni che sono specializzate nella sincronizzazione con Dropbox come FolderSync, Dropsync o cerca un'applicazione Google Play.
  • D. voglio sincronizzare MoneyManagerEx fro Android con altri servizi di cloud (Drive, Box.net, Wuala, ecc. ..). Come posso fare?
    R. In MoneyManagerEx per Android l'unico servizio di cloud storage disponibile al momento è Dropbox. Se si desidera utilizzare il database con altri servizi di cloud storage, ti consigliamo di individuare su Google Play l'applicazione che permette di sincronizzare con il vostro servizio. Il database verrà salvato nella memoria del dispositivo e quindi si potrà aprire con MoneyManagerEx for Android
  • D. Dopo una sincronizzazione più non vedo miei dati. Ho perso tutti i dati nel mio database?
    R. Tranquillo non hai perso nulla! In rari casi, quando si sincronizza il database non viene scaricato correttamente, e non è più utilizzabile per MoneyManagerEx for Android. In questi casi, si consiglia di forzare un download del database da Dropbox. Se ancora non funziona, è consigliabile dii eliminare i file in /MoneyManagerEx/dropbox/ e scaricare di nuovo il file.
  • D. è possibile ripristinare una versione precedente del database?
    R. Naturalmente si può fare. Dropbox mantiene tutte le versioni di tutti i tuoi file. Qui potete trovare la guida Dropbox su come ripristinare il file
