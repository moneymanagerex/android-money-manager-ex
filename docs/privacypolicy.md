# Privacy Policy for MoneyManagerEx for Android

**Last modified:** February 12, 2026

**MoneyManagerEx Project** ("us", "we", or "our") develops **Money ManagerEx for Android** (the "App"). This page informs you of our policies regarding the collection, use, and disclosure of Personal Information we receive from users of the App.

We use your information only for providing and improving the App. By using the App, you agree to the collection and use of information in accordance with this policy.

---

### Data Controller and Owner
**Money Manager Ex Project**
**Contact Email:** android@moneymanagerex.org

### Types of Data Collected

Among the types of Personal Data that this Application collects, by itself or through third parties, there are: **Device Information, Device ID, Database Path, Cloud Storage Name, and App Interactions, optionally log crash**.

* **Device Information:** Operating system version and device model.
* **Device ID:** Unique identifiers used for analytics.
* **Database Path:** The local file path where the database is stored on the device.
* **Cloud Storage Name:** The name of the service used for synchronization (e.g., Dropbox, Google Drive).
* **App Interactions:** Actions performed within the app, such as "start sync" or "stop sync".
* **Log Crash:** User can optionally add log crash in sharing page if an error occurs.

**Voluntary Nature of Data Collection** The collection of analytics data via Amplitude is **disabled by default**. The User must explicitly enable the "Send Anonymous Data" option in the App settings for this collection to occur. If the setting remains off, no analytics data is transmitted.
