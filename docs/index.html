Money Manager Ex for Android

Money Manager Ex is a free, open-source, cross-platform, easy-to-use personal finance software. It primarily helps organize one's finances and keeps track of where, when, and how the money flows

Desktop version of Money Manager Ex runs on all the major OS's, like Windows, MacOS, and Linux.

MoneyManagerEx for Android

Introduction

Money Manager Ex for Android is a port of the desktop personal finance application Money Manager Ex, currently available for Windows, Mac OSX and Linux. The Android version mirrors the best features of the desktop version while adding mobility and database synchronization.

In Money Manager Ex project, we strive to develop good personal finance software that we use ourselves. We also do this for fun and the love of programming. We are a non-profit, informal organization although we do accept donations and use them towards the hosting fees. Any leftovers for the Android version will also go towards acquiring a device for the app testing.

Features

All the feature requests can be tracked at our issue tracker on GitHub. .

Highlights

  • Opening databases by using storage access framework (SAF), allowing you to pick your favourite storage provider, whether cloud or local.
  • Transaction management
  • Multiple accounts and account types, currencies, and languages.
  • Account, payee, category, currency management
  • Budgets (view only, currently)
  • Investment watchlists
  • Database encryption
  • Attachments (view only, currently)
  • Security - application lock/unlock with pin code
  • Scheduled transactions
  • Reports and Charts

Testing - Pre-Release Versions

Your assistance with testing the pre-release (Beta) versions of the app is highly appreciated. This way you can directly help the project and ensure that the stable releases are bulletproof.

Should you want to try out the latest version of Money Manager Ex for Android, you can access it through F-Droid. Just enable pre-release versions and they will be listed in the version history but will not be offered as regular app updates. This allows easy reverting to previous versions should something go wrong.
