An easy-to-use personal finance application to manage personal finances, bank accounts, family budget, and so on.
The main goal is to add a mobile implementation for the MMEX Desktop application.

-------------------------------------------------
<b>Features</b>
------------------------------------------------

* Management of single transactions
* Account, payee, category, currency management
* Cloud Storage based synchronization, including
** cloud storage, e.g. Google Drive, Nextcloud
** sync service, e.g. Syncthing
* Compatible with MMEX for Desktop
* Scheduled transactions
* Charts for reports
* Supported languages: Bosnian, Chinese, Croatian, Czech, Dutch, English, German, Greek, Hebrew, Hungarian, Indonesian, Italian, Japanese, Polish, Portuguese, Russian, Spanish, Tagalog, Turkish, Ukrainian, Urdu, Vietnamese
