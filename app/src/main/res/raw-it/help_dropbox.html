MoneyManagerEx per Android

Sincronizzazione del database di MoneyManagerEx per Windows/Linux/Mac utilizzando il servizio Dropbox.

Prerequisiti: disporre di un account dropbox. Se ancora non lo si possiede, andare su www.dropbox.com e crearne uno nuovo.

  • Collega MoneyManagerEx per Android con Dropbox. Selezionare "Collega a Dropbox". Se sul dispositivo è già installata l'app di Dropbox, si apre automaticamente e richiede di autorizzare l'accesso a Dropbox per MoneyManagerEx. Se invece non è installata l'app di Dropbox sul tuo dispositivo, Android apre tramite browser la pagina di login di Dropbox. Dopo il login, sarà richiesto il permesso di autorizzare MoneyManagerEx per Android ad accedere al tuo account Dropbox.
  • Autorizzando MoneyManagerEx per Android, al primo accesso sarà creata automaticamente dal servizio Dropbox, la cartella / applicazioni/MoneyManagerEx Mobile all'interno del vostro spazio Dropbox. La cartella non deve mai essere creata manualmente!

    Uso già un database per la versione desktop
  • Se hai già un database (mmb) della versione Desktop MoneyManagerEx, devi copiare o spostare il file (s) all'interno della cartella /apps/MoneyManagerEx Mobile (si consiglia di effettuare questa operazione utilizzando il computer).
  • Dalle impostazioni di Dropbox del MoneyManagerEx for Android, cliccare su "File Dropbox". L'applicazione aprirà una nuova finestra con tutti i file nella cartella /apps/MoneyManagerEx Mobile. Seleziona il file che desideri connettere e torna indietro.
  • Una volta tornato alle impostazioni Dropbox di MoneyManagerEx for Android, dopo circa 10 secondi si avvierà il download del file selezionato, è possibile vedere la notifica nella barra di stato nel vostro dispositivo. Quando il download è terminato, il dispositivo ti avvisa che il file è pronto all'uso e si deve cliccare su "Apri Database" per completare il collegamento tra MoneyManagerEx for Android e Dropbox.

    • Non ho mai usato la versione desktop e voglio sincronizzare il database nel dispositivo
  • Se non avete già un file mmb di MoneyManagerEx versione Desktop, dovete cliccare su "Upload database su Dropbox." Dopo circa 10 secondi inizierà il caricamento del database in uso sul vostro spazio Dropbox. Alla fine del processo ci sarà la notifica del completamento dell'upload, e nella cartella /Apps/MoneyManagerEx Mobile troverete un file denominato data.mmb.

    • Altre opzioni
  • Dopo aver effettuato il collegamento all'account e al file, MoneyManagerEx per Android si sincronizza automaticamente con Dropbox ogni x minuti, come impostato nell'impostazione "Intervallo di sincronizzazione". Se non si desidera sincronizzare automaticamente impostare "Intervallo Sincronizzazione" a Mai. A questo punto la sincronizzazione dovrà essere effettuata manualmente.
  • È importante impostare "Caricare il database immediatamente dopo il cambiamento". Questa configurazione permette a MoneyManagerEx per Android di caricare il database su Dropbox immediatamente dopo aver apportato una modifica.
  • Utilizzare "Download database da Dropbox" o "Upload a Dropbox" per forzare l'operazione di Download o Upload del database, in caso di malfunzionamenti.
  • Quando hai collegato il tuo account Dropbox e il file, nel Drawer si attiva una nuova voce di menù che vi permette la sincronizzazione.

    • Domande frequenti
  • Q. Non riesco a collegarmi con Dropbox, o MoneyManagerEx non sincronizza?
    R. La prima cosa che devi fare è inviarci una mail android.money.manager.ex@gmail.com descrivendo il problema. Nel frattempo, se volete utilizzare MoneyManagerEx per Android si dovrà utilizzare altre applicazioni specializzate nella sincronizzazione con Dropbox come FolderSync, Dropsync o cerca un'applicazione Google Play.
  • Q. voglio sincronizzare MoneyManagerEx per Android con altri servizi di cloud (Drive, Box.net, Wuala, ecc. ..). Come posso fare?
    R. In MoneyManagerEx per Android l'unico servizio di cloud storage disponibile al momento è Dropbox. Se si desidera utilizzare il database con altri servizi di cloud storage, ti consigliamo di individuare su Google Play l'applicazione che permette di sincronizzare con il vostro servizio. Il database verrà salvato nella memoria del dispositivo e quindi si potrà aprire con MoneyManagerEx per Android
  • Q. Dopo una sincronizzazione più non vedo miei dati. Ho perso tutti i dati nel mio database?
    R. Non ti preoccupare, non hai perso nulla! In rari casi è possibile che, quando si sincronizza il database, che il file non venga scaricato correttamente e non sia più leggibile da MoneyManagerEx per Android. In questi casi, sarà necessario forzare un download del database da dropbox. Se il database ancora non si apre, si consiglia di eliminare i file in /com.money.manager.ex/dropbox/ e ritentare il download forzato dei file da Dropbox.
  • Q. Posso ripristinare una versione precedente del database?
    R. Ovviamente si può fare. Dropbox mantiene tutte le versioni di tutti i tuoi file. Qui potete trovare la guida Dropbox su come ripristinare il file
