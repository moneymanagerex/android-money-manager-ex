MoneyManagerEx pour Android

Synchronisation de base de données avec MoneyManagerEx pour Windows/Linux/Mac en utilisant le service Dropbox.

Prérequis : Avoir un compte Dropbox. Si vous n'en avez pas encore, allez sur www.dropbox.com pour en créer un nouveau.

  • Pour connecter MoneyManagerEx Android à Dropbox,. Sélectionnez "Associer à Dropbox". Si l'application Dropbox est installée sur votre appareil, elle s'ouvre automatiquement et vous demande d'autoriser MoneyManagerEx à accéder à votre Dropbox. Si l'application Dropbox n'est pas installée, votre navigateur s'ouvrira sur la page de connexion de Dropbox. Une fois connecté, on vous demandera d'autoriser MoneyManagerEx à accéder à votre Dropbox.
  • La première fois que vous autoriser MoneyManagerEx à accéder à votre Dropbox, un dossier /Applications/MoneyManagerEx Mobile sera automatiquement créé dans votre Dropbox. Ce dossier ne doit jamais être créé manuellement !

    J'utilise déjà la version bureau de MoneyManagerEx
  • Si vous avez déjà une version de bureau de la base de données (mmb) de MoneyManagerEx, vous devez copier ou déplacer le(s) fichier(s) dans le dossier / Applications/MoneyManagerEx Mobile (l'utilisation de l'ordinateur est recommandée pour cette opération).
  • Depuis les paramètres Dropbox de l'application MoneyManagerEx pour Android, cliquez sur "Fichier connecté à Dropbox". L'application ouvrira une nouvelle fenêtre avec tous les fichiers du dossier / Applications/MoneyManagerEx Mobile. Sélectionnez le fichier que vous souhaitez connecter puis revenez à l'écran précédent.
  • Environ 10 secondes après votre retour dans les paramètres Dropbox de MoneyManagerEx pour Android, le téléchargement du fichier sélectionné va commencer. Vous pourrez voir la notification qui apparaîtra sur votre appareil. Une fois le téléchargement terminé, l'appareil vous signale que le fichier est prêt et vous devez cliquer sur "Ouvrir la base de données" pour achever la connexion de l'application MoneyManagerEx pour Android et Dropbox.

    • Je n'ai jamais utilisé la version bureau et je veux synchroniser la base de données sur mon appareil
  • Si vous n'avez pas déjà un fichier mmb de la version bureau de MoneyManagerEx, cliquez sur « Envoyer vers Dropbox ». Dans les 10 secondes environ, l'envoi de base de données actuellement utilisée vers Dropbox va commencer. À la fin, une notification vous avertira de l'achèvement du transfert et un fichier "data.mmb" figurera dans le dossier / Applications/MoneyManagerEx Mobile.

    • Autres paramètres
  • Après avoir connecté votre compte Dropbox et le fichier, MoneyManagerEx pour Android synchronise automatiquement le fichier avec Dropbox toutes les x minutes définies dans « Intervalle de synchronisation ». Si vous ne souhaitez pas synchroniser automatiquement, vous pouvez régler l'intervalle de synchronisation sur zéro.
  • Il est important de cocher l'option «Envoyer immédiatement la base de données après chaque changement». Cette option permet à MoneyManagerEx d'envoyer la base de données vers Dropbox à chaque modification effectuée.
  • Utiliser «Télécharger la base de données depuis Dropbox» ou «Envoyer la base de données vers Dropbox» pour forcer les opérations de téléchargement ou de chargement de la base de données, notamment en cas de problème.
  • Lorsque vous avez connecté votre compte Dropbox et le fichier dans la fenêtre principale, activez la fonctionnalité de synchronisation dans la barre d’outils supérieure.

    • FAQ
  • Q. Je ne peux pas me connecter à Dropbox ou la synchronisation de MoneyManagerEx ne fonctionne pas ?
    R. La première chose à faire c'est de nous envoyer un courriel à android.money.manager.ex@gmail.com en nous décrivant le problème rencontré. En attendant, si vous voulez synchroniser votre base avec MoneyManagerEx, vous pouvez utiliser d'autres applications permettant de se synchroniser avec Dropbox comme FolderSync, Dropsync ou rechercher une application sur le Google Play.
  • Q. je veux synchroniser MoneyManagerEx avec d'autres services de cloud (Drive, Box.net, Wuala, etc. ..). Comment puis-je faire ?
    R. Le seul service de cloud actuellement disponible sur MoneyManagerEx pour Android est Dropbox. Si vous souhaitez synchroniser la base de données avec d'autres services de cloud, vous pouvez rechercher sur Google Play une application qui vous permet de vous synchroniser avec ce service. La base de données sera sauvegardées dans la mémoire externe de l'appareil et vous pourrez l'ouvrir avec MoneyManagerEx pour Android
  • Q. Après une synchronisation je ne vois plus mes données. Ai-je perdu toutes mes données ?
    R. Non, vous n'avez rien perdu ! Dans de rares cas, lorsque vous synchronisez la base de données, le fichier peut ne pas être téléchargé correctement, et il n'est donc plus utilisable par MoneyManagerEx pour Android. Dans ce cas, vous devez forcer le téléchargement de la base de données depuis dropbox. Si cela ne fonctionne toujours pas, nous vous recommandons de supprimer tous les fichiers du dossier /com.money.manager.ex/dropbox/ et de forcer à nouveau le téléchargement du fichier.
  • Q. Puis-je restaurer une ancienne version de la base de données ?
    R. Bien sûr. Dropbox conserve les anciennes versions de tous vos fichiers. Ici, vous pourrez trouver la procédure à suivre pour restaurer une ancienne version d'un fichier.
