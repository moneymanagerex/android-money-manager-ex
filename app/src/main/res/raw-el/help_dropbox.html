MoneyManagerEx για το Android

Συγχρονισμός Βάσης Δεδομένων με το MoneyManagerEx για Windows/Linux/Mac χρησιμοποιώντας την υπηρεσία του Dropbox.

Προϋπόθεση: να διαθέτεις ένα λογαριασμό στο dropbox. Εάν δεν διαθέτεις μπορείς να μεταβείς στο www.dropbox.com και να δημιουργήσεις ένα νέο λογαριασμό.

  • Συνδέστε το MoneyManagerEx για Android με το Dropbox. Επιλέξτε "Σύνδεση με το Dropbox." Εάν στη συσκευή σας έχει εγκατασταθεί η εφαρμογη του Dropbox, ανοίγει αυτόματα και σας ζητά να εγκρίνετε το MoneyManagerEx για Android για να αποκτήσει πρόσβαση στο Dropbox. Εάν δεν έχετε εγκαταστήσει την εφαρμογή του Dropbox στη συσκευή σας, το Android ανοίγει το πρόγραμμα περιήγησης στη σελίδα σύνδεσης του Dropbox. Αφού συνδεθείτε, θα σας ζητηθεί να επιτρέψετε στο MoneyManagerEx για Android να αποκτήσει πρόσβαση στο Dropbox σας.
  • If you authorize MoneyManagerEx for Android, the first time will be created automatically by Dropbox, the folder /Apps/MoneyManagerEx Mobile within your Dropbox space. Ο φάκελος ποτέ δεν πρέπει να δημιουργηθεί με το χέρι!

    Χρησιμοποιώ ήδη μια βάση δεδομένων για την desktop έκδοση (επιτραπέζιο η φορητό υπολογιστή )
  • If you already have a database (mmb) of MoneyManagerEx Desktop Version, you must copy or move the file (s) inside the folder /Apps/MoneyManagerEx Mobile (recommended use the computer for this operation).
  • Από τις ρυθμίσεις του Dropbox στο MoneyManagerEx για το Android, κάντε κλικ στην επιλογή "Dropbox συνδεδεμένο αρχείο". Application will open a new window with all the files in the folder /Apps/MoneyManagerEx Mobile. Select the file you want to connect and come back.
  • Once you are back in the Dropbox settings of MoneyManagerEx for Android, after about 10 seconds it will start downloading the selected file, you can see the notification that will open in your device. When the download is finished, your device notifies you that the file is ready to use and you must click on "Open Database" to complete the link between MoneyManagerEx for Android and Dropbox.

    • I've never used the desktop version and I want to synchronize the database on the device
  • If you do not already have a file of mmb MoneyManagerEx Desktop Version, you must click on "Upload to Dropbox database." After approximately 10 seconds the upload to Dropbox of the current database will start. At the end of the process there will be a notification for the completion of the upload, and in the /Apps/MoneyManagerEx Mobile you will find a file named data.mmb.

    • Other settings
  • Once you have made the link account and file, MoneyManagerEx for Android automatically syncs with Dropbox every x minutes set in the setting "Syncronization Interval". If you do not want to synchronize automatically set "Syncronization Interval" to zero.
  • It is important to set "Upload the database immediately after change". This setup enables MoneyManagerEx for Android to upload the database on Dropbox immediately after you make a change.
  • Use "Download database from Dropbox" or "Upload database to Dropbox" to force the operation to Download or Upload the database, in case of malfunctions.
  • When you linked your Dropbox account and the file in the main window activates the synchronization functionality on the top toolbar.

    • Συχνές Ερωτήσεις
  • Q. Δεν μπορεί να συνδεθεί με το Dropbox, ή το MoneyManagerEx δεν συγχρονίζει;
    A. The first thing you need to do is send us an email at android.money.manager.ex@gmail.com describing the problem. In the meantime, if you want to synchronize your database with MoneyManagerEx for Android, you will need to use other applications for this purpose, like FolderSync, Dropsync or search an application Google Play.
  • Q. I want to synchronize MoneyManagerEx for Android with other cloud services (Drive, Box.net, Wuala, etc. ..). How can I do?
    A. In MoneyManagerEx for Android the only cloud storage service available is Dropbox. If you want to use the database with other cloud storage services, you'll want to locate on Google Play the application that allows you to synchronize with your service. The database will be saved in the external memory of the device, and then open with MoneyManagerEx for Android
  • Q. After a sync I no longer see my data. I lost all the data in my database?
    A. Do not worry, you have not lost anything! In rare cases it is possible that, when you synchronize the database, the file is not downloaded properly and it is no longer readable by MoneyManagerEx for Android. In these cases, you will need to force a download of the database from dropbox. If the database still does not open, we recommend that you delete the files in /com.money.manager.ex/dropbox/ and retry the forced download of the files from Dropbox.
  • Q. Can I restore a previous version of the database?
    A. Φυσικά κάνω. Το Dropbox διατηρεί όλες τις εκδόσεις όλων των αρχείων σας. Εδώ μπορείτε να βρείτε τον οδηγό του Dropbox και στο πώς να επαναφέρετε το αρχείο
