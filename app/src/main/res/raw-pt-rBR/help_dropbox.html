MoneyManagerEx para Android

Sincronização da Base de Dados do MoneyManagerEx para Windows/Linux/Mac usando o serviço Dropbox.

Pré-requisito: ter uma conta dropbox. Se ainda não tem vá em www.dropbox.com e crie uma nova conta.

  • Conecte o MoneyManagerEx para Android com a Dropbox. Selecione "Conectar com Dropbox". Se o seu dispositivo tiver instalado o app Dropbox, ele abrirá automaticamente e pedirá para autorizar o MoneyManagerEx para Android acessar sua conta Dropbox. Se você não tiver o app Dropbox instalado no seu dispositivo, abra o navegador na página de login do Dropbox. Após o login, você deverá autorizar o MoneyManagerEx para Android acessar sua conta do Dropbox.
  • Se você autorizar o MoneyManagerEx para Android, pela primeira vez será criada automaticamente a pasta /Apps/MoneyManagerEx Mobile dentro do espaço da Dropbox. A pasta nunca deve ser criada manualmente!

    Eu já uso um banco de dados para a versão desktop
  • Se você já tiver um banco de dados (mmb) da versão de Desktop do MoneyManagerEx, você deve copiar ou mover o arquivo (s) dentro da pasta /Apps/MoneyManagerEx Mobile (recomendado usar o computador para esta operação).
  • As definições de Dropbox do MoneyManagerEx para Android, clique em "Dropbox linked file". O aplicativo abrirá uma nova janela com todos os arquivos na pasta /Apps/MoneyManagerEx Mobile. Selecione o arquivo que deseja conetar-se e volte.
  • Quando voltar ás configurações do Dropbox do MoneyManagerEx para Android, após 10 segundos ele vai começar a baixar o arquivo selecionado, você poderá ver a notificação que será aberta no seu dispositivo. Quando o download estiver concluído, o seu dispositivo notifica que o arquivo está pronto a ser usado, e deve clicar em "Abrir base de dados" para completar a ligação entre o MoneyManagerEx para Android e o Dropbox.

    • Eu nunca usei a versão desktop e quero sincronizar a base de dados no dispositivo
  • Se você ainda não tiver um arquivo mmb da versão Desktop do MoneyManagerEx, então será necessário clicar em "Enviar a base de dados para o Dropbox". Após aproximadamente 10 segundos começará a enviar a base de dados para o Dropbox atualmente em uso. Quando o processo de upload estiver completo, você será notificado, e na pasta /Apps/MoneyManagerEx Mobile terá um arquivo com o nome "data.mmb".

    • Outras configurações
  • Depois de ter feito a ligação da conta e do arquivo, o MoneyManagerEx para Android automaticamente sincroniza com o Dropbox a cada x minutos definidos na configuração "Intervalo de sincronização". Se não quiser sincronizar automaticamente defina "Intervalo de sincronização" para zero.
  • É importante definir para "Carregar o banco de dados imediatamente após a mudança". Esta configuração permite que o MoManagerEx para Android envie o banco de dados para o Dropbox imediatamente depois que você fizer uma alteração.
  • Use "Download da base de dados do Dropbox" ou "Envio para o Dropbox" para forçar a operação de Download ou Upload da base de dados, em caso de mau funcionamento.
  • Quando linkar a sua conta no Dropbox e o arquivo, na janela principal é ativada a funcionalidade de sincronização na barra de ferramentas superior.

    • Ajuda
  • P. Não consigo conectar-me ao Dropbox, ou o MoneyManagerEx não sincroniza?
    R. A primeira coisa que deve fazer é enviar-nos um e-mail para android.money.manager.ex@gmail.com descrevendo o problema. Enquanto isso, se quiser sincronizar a sua base de dados com o MoneyManagerEx para Android, precisará usar outra aplicação para essa finalidade, como o FolderSync, Dropsync ou procurar uma aplicação na Google Play.
  • P. Eu gostaria de sincronizar o MoneyManagerEx para Android com outros serviços na nuvem (Drive, Box.net, Wuala, etc. ...). Como posso fazer?
    R. No MoneyManagerEx para Android, o único serviço de armazenamento na nuvem disponível é o Dropbox. Se você deseja usar o banco de dados com outros serviços de armazenamento de nuvem, você deverá localizar no Google Play o aplicativo que permite que você sincronize com seu serviço. A base de dados será guardada na memória externa do aparelho e depois aberta com o MoneyManagerEx para Android
  • P. Após uma sincronização não consigo ver os meus dados. Perdi todos os dados na minha base de dados?
    R. Não se preocupe, você não perdeu nada! Em casos raros, é possível que, quando sincronizar a base de dados, não é feito o download corretamente do arquivo e então não fica legível pelo MoneyManagerEx para Android. Nesses casos, você precisa forçar um download da base de dados do dropbox. Se a base de dados ainda não abrir, recomendamos que remova os arquivos em /com.money.manager.ex/dropbox/ e tente novamente o download dos arquivos do Dropbox.
  • P. Posso restaurar uma versão anterior da base de dados?
    R. Claro que sim. Dropbox mantém todas as versões de todos os seus arquivos. Aqui você pode encontrar o guia do Dropbox em como restaurar o arquivo
