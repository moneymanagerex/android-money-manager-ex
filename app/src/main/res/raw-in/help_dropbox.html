MoneyManagerEx untuk Android

Sinkronisasi database dengan MoneyManagerEx untuk Windows/Linux/Mac menggunakan layanan Dropbox.

Prasyarat: memiliki akun dropbox. Jika Anda masih belum memilikinya ke www.dropbox.com dan buat akun baru.

  • Hubungkan MoneyManagerEx untuk Android dengan Dropbox. Pilih "Link to Dropbox." Jika perangkat Anda memasang aplikasi Dropbox, perangkat akan secara otomatis terbuka dan meminta Anda memberi otorisasi MoneyManagerEx untuk Android untuk mengakses Dropbox Anda. Jika Anda belum memasang aplikasi Dropbox di perangkat Anda, Android membuka browser di halaman masuk Dropbox. Setelah masuk, Anda akan diminta memberi otorisasi MoneyManagerEx untuk Android untuk mengakses Dropbox Anda.
  • Jika Anda memberi otorisasi MoneyManagerEx untuk Android, pertama kali akan dibuat secara otomatis oleh Dropbox, folder /Apps / MoneyManagerEx Mobile di dalam ruang Dropbox Anda. Folder tidak boleh dibuat secara manual!

    Saya sudah menggunakan database untuk versi desktop
  • Jika Anda sudah memiliki database (mmb) dari MoneyManagerEx Desktop Version, Anda harus menyalin atau memindahkan file (s) di dalam folder /Apps / MoneyManagerEx Mobile (disarankan gunakan komputer untuk operasi ini).
  • Dari pengaturan Dropbox MoneyManagerEx untuk Android, klik "file terkait Dropbox". Aplikasi akan membuka jendela baru dengan semua file di folder /Apps /MoneyManagerEx Mobile. Pilih file yang ingin Anda sambungkan dan kembali.
  • Setelah Anda kembali ke pengaturan Dropbox MoneyManagerEx untuk Android, setelah sekitar 10 detik, Anda akan mulai mendownload file yang dipilih, Anda dapat melihat pemberitahuan yang akan terbuka di perangkat Anda. Saat unduhan selesai, perangkat Anda akan memberi tahu Anda bahwa file tersebut siap digunakan dan Anda harus mengklik "Open Database" untuk melengkapi tautan antara MoneyManagerEx untuk Android dan Dropbox.

    • Saya tidak pernah menggunakan versi desktop dan saya ingin menyinkronkan database pada perangkat
  • Jika Anda belum memiliki file mmb MoneyManagerEx Desktop Version, Anda harus mengklik "Upload to Dropbox database." Setelah kira-kira 10 detik upload ke Dropbox dari database saat ini akan dimulai. Pada akhir proses akan ada pemberitahuan untuk penyelesaian upload, dan di /Apps / MoneyManagerEx Mobile Anda akan menemukan sebuah file bernama data.mmb.

    • Pengaturan lainnya
  • Setelah membuat akun dan file tautan, MoneyManagerEx untuk Android secara otomatis disinkronkan dengan Dropbox setiap x menit yang ditetapkan dalam setelan "Interval Sinkronisasi". Jika Anda tidak ingin melakukan sinkronisasi secara otomatis tetapkan "Interval Sinkronisasi" menjadi nol.
  • Penting untuk mengatur "Upload database segera setelah perubahan". Penyiapan ini memungkinkan MoneyManagerEx untuk Android mengunggah database di Dropbox segera setelah Anda melakukan perubahan.
  • Gunakan "Download database dari Dropbox" atau "Upload database ke Dropbox" untuk memaksa operasi Download atau Upload database, jika terjadi malfungsi.
  • Saat Anda menghubungkan akun Dropbox Anda dan file di jendela utama akan mengaktifkan fungsi sinkronisasi di toolbar atas.

    • FAQ
  • Q. Saya tidak bisa terhubung dengan Dropbox, atau MoneyManagerEx tidak sinkron?
    A. Hal pertama yang perlu Anda lakukan adalah mengirimi kami email di android.money.manager.ex@gmail.com yang menjelaskan masalahnya. Sementara itu, jika Anda ingin menyinkronkan database dengan MoneyManagerEx untuk Android, Anda perlu menggunakan aplikasi lain untuk tujuan ini, seperti FolderSync,Dropsync atau cari aplikasi Google Play.
  • Q. aya ingin menyinkronkan MoneyManagerEx untuk Android dengan layanan awan lainnya (Drive, Box.net, Wuala, dll. ..). Bagaimana saya bisa melakukannya?
    A. Di MoneyManagerEx untuk Android, satu-satunya layanan penyimpanan awan yang tersedia adalah Dropbox. Jika Anda ingin menggunakan database dengan layanan penyimpanan awan lainnya, Anda ingin mencari di Google Play aplikasi yang memungkinkan Anda melakukan sinkronisasi dengan layanan Anda. Database akan disimpan di memori eksternal perangkat, dan kemudian dibuka dengan MoneyManagerEx untuk Android
  • Q. Setelah sinkronisasi saya tidak lagi melihat data saya. Saya kehilangan semua data di database saya?
    A. Jangan khawatir, kamu belum kehilangan apapun! Dalam kasus yang jarang terjadi, mungkin saja ketika Anda menyinkronkan database, file tersebut tidak dapat didownload dengan benar dan tidak dapat dibaca lagi oleh MoneyManagerEx untuk Android. Dalam kasus ini, Anda perlu memaksa download database dari dropbox. Jika database tetap tidak terbuka, sebaiknya hapus file di /com.money.manager.ex/dropbox/ dan coba lagi download file dari Dropbox paksa.
  • Q. Dapatkah saya mengembalikan versi database sebelumnya?
    A. Tentu saja. Dropbox menyimpan semuanya versi dari semua file Anda. Di sini Anda bisa menemukan panduan Dropbox tentang cara mengembalikan file
